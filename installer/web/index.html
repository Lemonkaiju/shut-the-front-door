⚙️ Advanced: Agentic AI Engineering

For technical users who want to go further and set up a local AI coding assistant (like Antigravity or Windsurf) on their privacy stack.

What is an Agentic AI?

An agentic AI is an AI assistant that can autonomously read, write, and execute code on your machine. Think of it as a pair programmer that lives locally on your hardware, with no data leaving your network.

Requirements

  • Hardware: 16GB+ RAM, modern CPU (or GPU with 8GB+ VRAM)
  • OS: Linux (Bazzite/LemonKaijuOS recommended) or Windows with WSL2
  • Ollama: Already installed by the AI Setup step above
  • Model: qwen2.5-coder:14b or deepseek-coder-v2:16b for coding tasks

Setup Steps

  1. Ensure Ollama is running: ollama serve
  2. Pull a coding model: ollama pull qwen2.5-coder:14b
  3. Install Windsurf IDE or Cursor
  4. Point the AI backend to http://localhost:11434 (Ollama API)
  5. Install agent frameworks: pip install langchain autogen
  6. Clone Antigravity or your preferred agent framework

Privacy Note

When running a local model via Ollama, no code or prompts leave your machine. This is the only truly private AI coding assistant setup.

Recommended Models

qwen2.5-coder:14b ~9GB VRAM Best for most users
deepseek-coder-v2:16b ~10GB VRAM Advanced coding
llama3.1:8b ~5GB VRAM Low-spec fallback
